hexo基础命令

2021-10-23 152 字数
# hexo

欢迎来到Hexo!这是你的第一篇文章。查看文档了解更多信息。如果您在使用Hexo时遇到任何问题,可以在疑难解答中找到答案,也可以在GitHub上询问我。

快速启动😏

创建新帖子

$ hexo new "My New Post"

可以简写为$ hexo n

运行服务

$ hexo server

可以简写为$ hexo s

生成静态文件

$ hexo generate

可以简写为$ hexo g

部署到远程站点

$ hexo deploy

可以简写为$ hexo d

清理生成的动态文件

$ hexo clean

当修改没有生效时可以使用此命令
